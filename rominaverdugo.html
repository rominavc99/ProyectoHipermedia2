ROMINA VERDUGO

Desarrollo del HTML y CSS base para el proyecto, así como perfeccionamiento de detalles. Desde el DOCTYPE hasta la vinculación con el JavaScript y dando los estilos básicos para el display correcto de los elementos en nuestra pantalla, que después servirá como base para seguir agregando elementos clave y mejorando el contenido. Detallando mejor, con la base del HTML me refiero al header con todas sus etiquetas, la vinculación del CSS, el recortado y exportación a png de la imagen principal, así como su inclusión en el index y los botones principales del afinador, así como la vinculación con el JavaScript, como mencioné con anterioridad. Básicamente fue el esqueleto del sito, el cuál sirvió como guía para seguir trabajando, ya que yo comencé el proyecto.
En el CSS, trabajé los estilos para todos los elementos que mencioné con anterioridad, como establecer las posiciones de los objetos, ajustes a la tipografía, los botones, y la imagen, de manera que funcionaran bien en conjunto.
Fueron bastantes detalles puesto que en el CSS siempre hay muchas propiedades, pero todos tenían el objetivo de contribuir a una mejor estructura para el sitio. En general mi trabajo se basó en la maquetación base del sitio principalmente, así como en el apoyo para otras áreas, toda mi idea se basaba en lograr las bases para que el proyecto pudiera llegar al resultado más atractivo e intuitivo posible para nuestros usuarios, haciendo uso de todas las habilidades que tenemos hasta el momento.
